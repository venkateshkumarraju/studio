# **App Name**: GeoEngine Pro

## Core Features:

- Headline and Value Proposition: Display a compelling headline and subheadline emphasizing your value as a GIS engineer.
- Skills Showcase: Showcase key skills with visual progress bars or icon-based representations.
- Projects Portfolio: Present projects with images, descriptions, and links to live demos or code repositories.
- Contact Information: Display contact information (email, phone) and links to professional profiles (LinkedIn, GitHub).
- Call-to-Action: Include a prominent 'Request a Consultation' button with a contact form submission.
- Resume Download: Offer a downloadable PDF version of your resume.

## Style Guidelines:

- Primary color: Deep teal (#008080) to convey expertise and trustworthiness.
- Background color: Light teal (#E0F8F8) for a clean and professional feel.
- Accent color: Light orange (#E59500) for CTAs and visual interest.
- Headline font: 'Poppins', sans-serif, for a geometric, modern feel. Body Font: 'PT Sans', sans-serif, to maintain readability.
- Use a clean, single-column layout for easy readability on all devices.
- Employ a consistent set of professional, minimalist icons to represent skills and technologies.
- Implement subtle animations, such as fade-ins and smooth scrolling, to enhance user engagement.
